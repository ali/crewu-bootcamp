Looking and Moving around!

note: commands have 3 parts, the command name, the options (given by adding a “-” before them), and any arguments the command takes

Example:

ls -lah /home/goodwin/homework

This is the command ls, with the options l, a, h given the argument: /home/goodwin/homework

Things listed in square brackets [like this] are optiona

To see what is in a given folder use the command “ls”. By default it lists the contents of the folder you are in, but by changing the options, and giving it a file path, you can list the contents of other folders, and more information about those contents.

Command: ls [options] [$path]

Purpose: Display the contents of the directory

Relevant Option

Result

-a

Reveals all (including hidden)

-l

Reveals permissions, timestamps etc.

--color=auto

Colorizes the output, easier on the eyes

-h

Formats information to “human readable” (best used for sizes)

<example ls images>

Once you have looked around what about changing directories? Use the command “cd” to change the directory you are in, to another one.

Command: cd destination

Purpose: move you from your current directory to destination

Takes either a relative path, or an absolute path.

Command: mkdir[-options] [$path]

Purpose: Display the contents of the directory

Relevant Option

Result

-p

Make all intermediate directories as necessary

-v

Verbose, list directories as created

--color=auto

Colorizes the output, easier on the eyes

-h

Formats information to “human readable” (best used for sizes)
